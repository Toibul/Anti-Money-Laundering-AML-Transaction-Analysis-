# Anti-Money-Laundering-AML-Transaction-Analysis-
A data analytics project focused on detecting suspicious financial transactions using SQL for data analysis and Power BI for interactive dashboards. The project explores transaction patterns, high-risk transfers, cross-border activities, and potential money laundering indicators.

This project I analyzes financial transaction data to identify potential money laundering activities using SQL for data exploration and Power BI for visualization.

The goal is to simulate how financial institutions analyze transactions to detect suspicious patterns, high-risk transfers, and cross-border activities.

🛠 Tools & Technologies
SQL – Data cleaning and analysis
Power BI – Data visualization and dashboard

Data source : https://www.kaggle.com/datasets/berkanoztas/synthetic-transaction-monitoring-dataset-aml
